# Kiera Theme for Hugo

## Installation

```bash
cd themes
git clone ... kiera
```
Using ```git submodule``` is recommended instead of ```git clone``.

## Configuration

Optional ```config.toml```:

```toml
pygmentsCodeFences = true

disqusShortname = ""
googleAnalytics = ""

[author]
    name = ""
    github = ""
    gitlab = ""
    linkedin = ""
    facebook = ""
    twitter = ""
    instagram = ""

[params]
    tagline = "the tagline for this website"

```
